Unzip and Place the FlipSmartHome library folder in your PC path: <arduinosketchfolder>/libraries/ folder. You may need to create the libraries subfolder if its your first library. Restart the IDE.

Another way is using the arduino IDE (V1.6.x and up)
Sketch --> Include Libraries --> Add .ZIP Libraries
Select the .ZIP library and open.
